# KWA Science + Biology + Chemistry Curriculum Module

This is a static, GitHub Pages-ready curriculum module for Science + Biology + Chemistry.

## What it includes

- Basic, Intermediate, and Advanced levels
- Original lesson content based on source-book topics
- Homework tasks and rubrics for every lesson
- Five-question lesson quizzes
- Forty-question mastery tests per level
- Auto-scoring
- Correct answer explanations
- Recommended review tags
- Progress tracking with `localStorage`
- Exportable progress JSON

## File map

```text
index.html
styles.css
src/app.js
src/curriculum.js
src/questions.js
```

## How to use

1. Upload the folder to GitHub.
2. Turn on GitHub Pages.
3. Open the site.
4. Start with Basic, then Intermediate, then Advanced.

## How to merge into an existing site

- Copy `src/curriculum.js` into your main data file or import it.
- Copy `src/questions.js` into your question-bank file or import it.
- Copy the assessment logic from `src/app.js` into your existing app router.
- Copy the CSS blocks into your existing stylesheet.
